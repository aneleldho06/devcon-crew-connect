

# TechFest 2026 Volunteer Management System — Landing Page

## Overview
A dark-themed, modern landing page that introduces and links to the Engelsystem volunteer management tool.

## Pages & Structure
Single-page design with smooth scroll navigation across these sections:

### Navigation
- Fixed top navbar with logo text "TechFest 2026" and anchor links to each section
- Mobile hamburger menu

### Hero Section
- Large title "TechFest 2026 Volunteer System" with subtitle
- Two CTA buttons: "Open Volunteer System" (links to localhost:5080, new tab) and "Learn More" (scrolls to About)
- Subtle gradient or glow background effect

### About Section
- Brief explanation of Engelsystem and its purpose
- Clean card or text layout

### Features Section
- 4 feature cards with icons: Registration, Shift Scheduling, Role Management, Admin Dashboard
- Grid layout, subtle hover effects

### How It Works
- 4-step numbered flow with icons: Sign Up → Browse Shifts → Select Tasks → Admin Manages

### Demo Access Section
- Display demo credentials (Admin & Volunteer) in styled cards
- "Open Volunteer System" button

### Technical Overview
- Brief Docker/MariaDB deployment info in a minimal card

### Footer
- Project description, "Built for Engelsystem Deployment Quest", name placeholder

## Design
- Dark theme using the existing dark mode CSS variables
- Clean typography, generous spacing
- Lucide icons for features
- Subtle fade-in animations on scroll using CSS or intersection observer
- Fully responsive (mobile-first)
